font 태그

속성들 : face, size, color 등등
 
size 속성 : 1~7까지 지정 가능(숫자가 클수록 큰 글씨)
face 속성 : 원하는 글꼴 지정. 2가지 이상은 콤마(,)로 구분
기본 글자 크기는 basefont 태그를 이용하여 지정
+나 -를 이용하여 글자 크기 상대적으로 지정 가능
